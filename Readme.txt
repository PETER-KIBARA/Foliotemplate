# Foliotemplate

**FolioOne** is a sleek, static portfolio website template powered by [BootstrapMade.com](https://bootstrapmade.com/folioone-bootstrap-portfolio-website-template/).

---

## About

This repository provides a modern portfolio template built with Bootstrap, ideal for showcasing your projects, services, resume, and more. It includes multiple ready-to-use HTML pages and assets for quick setup.

## Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/PETER-KIBARA/Foliotemplate.git
   cd Foliotemplate
   ```

2. **Open locally**
   - Double-click `index.html` or open it in your web browser.

3. **Optional: Serve locally with Python**
   ```bash
   python -m http.server 8000
   ```
   Then visit [http://localhost:8000](http://localhost:8000) in your browser.

## Usage

- Customize the HTML pages to reflect your own content and branding.
- Update or replace assets (CSS, images, fonts, etc.) in the `assets` folder.
- Modify or remove sections (e.g., portfolio details, services) as needed.
